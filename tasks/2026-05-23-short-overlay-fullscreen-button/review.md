## 1. 手测脚本（四处入口都跑一遍）

### 1.1 短视频搜索浮层

1. 打开 App → 进入「搜索」tab → 输入关键词搜出至少 3 条短视频
2. 点击第 2 条进入播放浮层
3. **✓ 右侧操作栏底部出现全屏按钮**（在「播放模式」按钮下方）
4. 验证按钮 contentDescription 长按可读出「全屏播放」
5. 点击全屏按钮：
   - **✓ 屏幕立即旋转到 LANDSCAPE**（300ms 内）
   - **✓ 视频从原位置接续播放**（不黑屏、不卡顿、不从头开始）
   - **✓ 状态栏 / 导航栏隐藏**
   - **✓ `LongFormVideoPlayer` 控制条出现**：进度条、当前/总时长、暂停/继续、字幕入口
6. 验证手势：
   - 单击屏幕 → 控制条显示/隐藏切换
   - 5 秒无操作 → 控制条自动隐藏
   - 进度条上拖动 → seek 正常
   - 屏幕左半垂直拖拽 → 亮度调节，中央反馈
   - 屏幕右半垂直拖拽 → 音量调节，中央反馈
   - 水平拖拽 → 显示快进/快退预览
   - 长按 → 倍速播放
7. **✓ 视频播完后单循环重播**，不切到下一条
8. 点击右下「退出全屏」icon：
   - **✓ 屏幕旋转回竖屏**
   - **✓ 系统栏恢复**
   - **✓ 回到原浮层、停在原视频，继续播放（不暂停）**
   - **✓ 右上「顺序播放/单循环」状态恢复为用户原偏好**
9. 重新进入全屏 → 这次用系统返回键退出：
   - **✓ 第一次按返回 → 退出全屏，回到浮层（不关闭浮层）**
   - **✓ 第二次按返回 → 关闭整个搜索播放浮层**

- 字幕轨道为空时 `LongFormVideoPlayer` 的字幕入口视觉是否合理（如不合理，下一个迭代补 `showSubtitleButton: Boolean = false` 参数）
